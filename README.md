# Automated Business Reporting Pipeline

This project demonstrates an end-to-end data pipeline that:

- Extracts data from an external API
- Transforms it using Python
- Stores it in a database
- Automates reporting via n8n

## Tech Stack
- Python
- SQLite
- n8n

## Pipeline Flow
API → Python → Database → Automation → Email Report

## Purpose
Designed to showcase data automation and ETL skills for freelance work.
